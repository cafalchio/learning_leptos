# Welcome to Leptos Rust and Tailwind Learning Website

## Overview

This website is designed to help you get started with Leptos Rust, a lightweight web framework for Rust, and Tailwind CSS, a utility-first CSS framework. Whether you're a beginner or an experienced developer, this resource aims to provide a smooth learning experience.

## Getting Started

### Prerequisites

Before diving into Leptos Rust and Tailwind, make sure you have the following installed:

- Rust: [Install Rust](https://www.rust-lang.org/learn/get-started)

- npm: [Install npm](https://www.npmjs.com/get-npm)

### Setup

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/leptos-tailwind-learning.git
   ```

2. Navigate to the project folder:

   ```bash
   cd leptos-tailwind-learning
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Build the Rust project:

   ```bash
   cargo build
   ```

5. Start the development server:

   ```bash
   npm run start
   ```

6. Open your browser and go to [http://localhost:3000](http://localhost:3000) to see the website in action!

- `src/`: Rust source code for the Leptos Rust application.

- `static/`: Static assets for the website.

- `styles/`: Tailwind CSS styles and configurations.

- `index.html`: Main HTML file.
